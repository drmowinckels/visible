## [0.1.0] - 2026-05-05

Initial public release.

### Added

- In-browser analysis of Visible app CSV exports via WebR
- Six chapters covering pacing, recovery, sleep, symptoms, notes, and anomalies
- Sample multi-year dataset for exploring without uploading personal data
- CC BY 4.0 license
- Page footer with free-tier notice and version information
